Img Search Abstraction API

Coded by Jessica Peck

How To Use:

  • To search for images, go to: https://jp-img-search.herokuapp.com/api/search/(your-search-term)

  • To view different pages, add ?offset=(your-page-number) to the end of the search url

  • To view recent searches, go to: https://jp-img-search.herokuapp.com/api/recent
